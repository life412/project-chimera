# Project Chimera Day 2 Repository

This repository contains the deliverables for Day 2 of Project Chimera. It is organized for clarity, modularity, and ease of navigation.

## Structure

- **SRS/**: System Requirements Specification and related documents.
- **specs/**: Agent governance, operational rules, persona templates, and genesis prompts.
- **skills/**: Documentation of MCP servers, tools, and libraries referenced in the SRS.
- **research/**: Summaries of key protocols, architectures, and methodologies.
- **rules/**: Agent rules and operational guidelines.

## Submission Info

- All files are for documentation and planning only. No code is included.
- Internal links connect SRS, specs, and research for easy reference.
